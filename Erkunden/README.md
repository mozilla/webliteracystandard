Web Literacy Standard
=====================

The Exploring strand is made up of the following competencies:
'Erkunden' ist von der folgende Kompetenzen abgezeichnet:


* Navigation - using software tools to browse the Web
Navigation - die Benutzung von Software und Werkzeuge um im Web zu bewegen.

* Web Mechanics - understanding the Web ecosystem
Web Mechanik - verstehung des Webs Ökosystem
* Search - locating information, people and resources via the Web
Suchen - Information, Leute und Ressourcen im Web finden
* Credibility - critically evaluating information found on the Web
 - kritische Evaluierung von Information im Web
* Security - keeping systems, identities, and content safe 
Securität - Systeme, Identitäten und Inhalte sicher im Web halten können
